# SunPass Customer Service Performance Analysis (Power BI)

## Overview
This project analyzes SunPass customer service operations from August to October 2025 using Power BI.
It focuses on call volume trends, service efficiency, and customer satisfaction.

## Tools Used
- Power BI
- Microsoft Excel
- Microsoft Word

## Key Metrics Analyzed
- Calls Offered vs Calls Answered
- Average Speed of Answer (ASA)
- Average Handle Time (AHT)
- Abandon Rate
- Customer Satisfaction (CSAT)

## Key Insights
- Call volume increased month-over-month with no service degradation
- ASA improved significantly in October
- Abandon rate decreased and approached SLA targets
- CSAT improved consistently across all months
- Lower ASA directly correlated with higher CSAT

## Files Included
- Power BI Dashboard (.pbix)
- Raw Data (Excel)
- PDF Dashboard Export
- Performance Analysis Report

## Outcome
This dashboard enables leadership to track performance, identify peak periods, and make data-driven operational decisions.
